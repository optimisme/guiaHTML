Per fer llistes, es pot fer servir el tag <ul>, permet definir llistes amb cercles. Cada element de la llista ha d'estar dins del tag <li>.

  • Element A
  • Element B
  • Element C

Per fer llistes numerades, es pot fer servir el tag <ol>, que permet definir llistes amb números. Cada element de la llista ha d'estar dins del tag <li>.

  1. Element A
  2. Element B
  3. Element C

Per fer llistes definides, es pot fer servir el tag <dl>. Cada element de la llista ha d'estar dins dels tags <dt> o <dd>.

Black hot drink
Strong taste
Milk
White cold drink
Mixable with tea
